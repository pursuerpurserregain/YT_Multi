## 🔧 Configuration

Edit `config.yaml` to customize your settings:

```yaml
# config.yaml
general:
  threads: 50
  timeout: 30
  retry_attempts: 3

proxy:
  enabled: true
  rotation_interval: 60
  type: "http"  # http, socks4, socks5
  file: "proxies.txt"

view_bot:
  watch_time_min: 30    # seconds
  watch_time_max: 300   # seconds
  daily_limit: 10000
  human_behavior: true

like_bot:
  delay_min: 5
  delay_max: 15
  daily_limit: 5000

subscriber_bot:
  delay_min: 10
  delay_max: 30
  daily_limit: 1000

anti_detection:
  fingerprint_spoofing: true
  user_agent_rotation: true
  canvas_noise: true
  webgl_noise: true
```
